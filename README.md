# Demo app with Rails 4 / AngularJS and Github API

[AngularJS](http://angularjs.org/)

[Github API](https://developer.github.com/)

## HowTo

        git clone https://github.com/madwork/angular-github angular-github
        cd angular-github
        
        bundle install
        bundle exec rails s
